# design tokens

## spacing
- xxs: 4
- xs: 8
- sm: 12
- md: 16
- lg: 24
- xl: 32
- xxl: 40

## corner radius
- sm: 6
- md: 10
- lg: 14
- xl: 18

## stroke
- subtle: 1

## shadow
- use sparingly
- prefer no shadow
- if needed: very soft, low opacity, small blur

## layout widths
- compact panel max width: 320
- settings content max width: 720
- reading content max width: 800

## typography
- title: bold, compact, high contrast
- section header: semibold
- body: regular
- caption: slightly muted
- avoid too many font sizes
- rely on weight and spacing more than dramatic size jumps

## color behavior
- follow system appearance
- use system background layers
- accent color only for:
  - selected state
  - primary action
  - focus
- destructive actions should be clearly separated
- secondary text should be muted but readable

## dividers
- prefer subtle separators
- use spacing first
- only add divider when grouping is unclear without it

## animation
- fast
- subtle
- functional
- no playful bounce unless product explicitly calls for it
