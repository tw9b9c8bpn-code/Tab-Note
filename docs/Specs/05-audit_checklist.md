# audit checklist

- does spacing follow token scale exactly
- are there any magic numbers
- is there only one dominant primary action per area
- are unrelated controls separated
- does the selected sidebar row feel clear but restrained
- are cards consistent in padding and radius
- is the UI calm at first glance
- does any element feel louder than necessary
- does the screen still look good when text wraps
- does it work in light and dark mode
