# product intent

## purpose
Build software that feels calm, sharp, modern, and low-friction.

## principles
- hierarchy through spacing before color
- clarity over decoration
- one primary action per area
- subtle grouping, not heavy borders
- dense enough for power users, clean enough for normal users
- default to quiet UI
- reduce cognitive load
- consistency beats novelty

## emotional target
The interface should feel:
- composed
- premium
- effortless
- precise
- trustworthy

## anti-goals
Avoid:
- loud gradients
- excessive shadows
- random corner radii
- too many accent colors
- crowded layouts
- unnecessary labels
- over-explaining in the UI
