HTTPSSE / NDJSON
Edit flow: ContentView binds the selected note into NoteEditorView, which pushes text + RTF mutations back through NotesStore into SwiftData.
AI flow: Editor trigger reads the current paragraph, snapshots AI settings, streams through AIService, then renders into the shared AI panel.
Window flow: Tab drag updates NotesStore state, AppDelegate creates / closes detached panels, and moving a note between windows preserves its note identity.
Update flow: AppDelegate checks AppUpdater on launch, and the status/menu path reports whether Sparkle is production-configured.
